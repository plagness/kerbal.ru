# Planet texture sources

All textures are stored locally so the page does not depend on third-party CDNs at runtime.

- `earth-blue-marble.jpg` — NASA Visible Earth, [Blue Marble: Land Surface, Ocean Color and Sea Ice](https://visibleearth.nasa.gov/images/57730/the-blue-marble-land-surface-ocean-color-and-sea-ice).
- `earth-clouds.jpg` — NASA Visible Earth, [Blue Marble: Clouds](https://visibleearth.nasa.gov/collection/1484/blue-marble).
- `mars-viking.jpg` — [Viking global mosaic](https://space.jpl.nasa.gov/tmaps/mars.html), Caltech/JPL/USGS.
- `phobos-viking.jpg` — [Viking/USGS texture map](https://space.jpl.nasa.gov/tmaps/mars.html), Caltech/JPL/USGS.
- `deimos-viking.jpg` — [Viking/USGS texture map](https://space.jpl.nasa.gov/tmaps/mars.html), Caltech/JPL/USGS.
- `moon-lroc.jpg` — [LROC WAC color mosaic](https://svs.gsfc.nasa.gov/4720/), NASA Scientific Visualization Studio.
- `moon-elevation.jpg` — [LOLA elevation map](https://svs.gsfc.nasa.gov/4720/), NASA Scientific Visualization Studio.

NASA imagery is generally not subject to copyright in the United States; NASA/JPL credit is retained here and in the site footer. See the linked source pages for their usage notes.
