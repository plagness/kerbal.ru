# Rocket model source

- `saturn-v-nasa.glb` — Saturn V, NASA/Michael D. Carbajal ([NASA 3D Resources](https://science.nasa.gov/3d-resources/saturn-v/), [NASA GitHub](https://github.com/nasa/NASA-3D-Resources/tree/master/3D%20Models/Saturn%20V)).

The original GLB is loaded locally and split at runtime into four independently controlled sections. Three spent stages detach during the one-way scroll mission; the upper spacecraft continues to Mars.

NASA's 3D Resources hub describes its assets as free to download and use. Refer to the linked NASA media usage guidelines for detailed terms.
