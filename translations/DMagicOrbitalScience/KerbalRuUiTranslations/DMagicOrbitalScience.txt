# DMagicOrbitalScience -- словарь UI-строк, извлечённых из скомпилированного DMagic.dll
# (DMagic/DMagicOrbitalScience 1.4.3, https://github.com/DMagic1/Orbital-Science).
# UI мода ЗАШИТ В КОД (хардкод английских литералов), НЕ через Localizer/#LOC:
#   monodis показал ссылки на KSP.Localization/Localizer, но 0 токенов #LOC/#autoLOC в куче
#   строк -- сами подписи окон/кнопок/part-меню/контрактов -- обычные English-литералы.
#   Перехватываются Harmony-патчами tools/ui-translator (GUI/GUILayout, UnityEngine.UI.Text,
#   TMPro). Часть научных данных (титулы/описания экспериментов, детали) переведена отдельно
#   через ModuleManager-.cfg (translations/DMagicOrbitalScience/Localization/, метод mm-*).
# Извлечение: monodis --userstrings DMagic.dll + прямой разбор кучи #US (UTF-16LE, обе
#   выравнивания байт), отбор только статичных player-facing строк.
# Формат: Английский<TAB>Русский. Несовпадение = строка остаётся как есть (no-op).
#
# СОЗНАТЕЛЬНО НЕ ВКЛЮЧЕНО (не пропуск, а решение):
# - Строки с рантайм-токенами ({0}, {0:P0}, @, <<1>>, N0/F1-форматы, регэкспы) -- собираются
#   на лету, точного совпадения не дадут.
# - Лог/диагностика ([DM]/[DMOS] ..., "Something Went Wrong", "Failed To Load ...",
#   "Could not find ...", "Stupid ...", "Door Invoked") -- не UI игрока.
# - Титулы-экспериментов как строки-КОНСТАНТЫ поиска ("Anomaly Scan", "Magnetometer Scan",
#   "Radio Plasma Wave Scan", "Recon Scan", "SIGINT Scan") -- отображаемые титулы уже
#   локализованы через науку-.cfg; эти литералы в DLL служат для сопоставления в контрактах,
#   их подмена рискует тихо сломать выдачу/учёт контрактов.
# - Имена аномалий / классы астероида / стороны света ("North", "Class A" и т.п.) -- в UI
#   попадают только как {n}-подстановка внутри динамических строк, самостоятельно не рисуются.
# - Тело диалога удаления данных ("Removing the experiment data will render this module
#   inoperable.\n\nRestoring functionality will require a Scientist.") -- содержит буквальные
#   переносы строки \n\n, невыразимо в построчном TAB-формате; пропущено намеренно.
#
# Политика: максимум по-русски; латиницей бренды (DMagic) и коды; «ёлочки», ё.
