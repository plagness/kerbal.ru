# HaystackContinued (Haystack ReContinued, https://github.com/aederdebais/HaystackContinued)
# — словарь UI-строк с ХАРДКОД-текстом: подписи зашиты в HaystackReContinued.dll (не в Localizer),
# извлечены `monodis --userstrings HaystackReContinued.dll`.
# Окно переключения/поиска аппаратов Haystack: список аппаратов, фильтры типов,
# сортировка, окно информации об аппарате и о небесном теле, стыковочные узлы.
#
# Движок kerbalru-ui-translator патчит GUI/GUILayout/GUIContent/TMP set_text и делает ТОЧНОЕ
# ordinal-совпадение всей строки (Dict.TryGetValue). Поэтому включены только строки, попадающие
# в виджет ЦЕЛИКОМ как отдельный литерал: подписи полей с завершающим ": " (рендерятся отдельными
# GUILayout.Label в горизонтальных группах), тултипы кнопок, заголовки окон/секций, статусы, кнопки.
#
# Опечатки английских оригиналов сохранены В КЛЮЧАХ намеренно (иначе не совпадёт):
#   "Vessel Infomation", "Orbial information:", "Atmopshere Ends: ".
# Ведущие/хвостовые ПРОБЕЛЫ и двоеточия подписей полей сохранены точно.
#
# Отброшено как не-UI (~171 непустая строка): имена классов/методов/событий
# (DeployedScienceController, RequestVessel, IRFetchVesselList, NavyFish.ModuleDockingNodeNamed),
# ключи конфига (window_positions, bottom_buttons, type_visibility), пути/файлы
# (settings.cfg, GameData/HaystackContinued, *.png-иконки, {0}/toolbar_icon), #autoLOC-ключи,
# hex-цвет #f4eac0, SI-префиксы (k/M/G/m/n/km/kg/N...), format-обрывки ({0} {1}, {0}kPa ({1}atm),
# 00.{0}, N0/F2/0.###E+0), лог-строки (Exception caught..., Deleting old settings file.,
# "vessel type: "+" is invalid.", HiddenVessles#Save: exception:), односимвольные латинские
# коды-кнопки (X/R/T/NB — глобальный словарь → коллизии), бренд-строки без перевода
# ("Haystack ReContinued {0}"), а также рантайм-конкатенированные сообщения, которые целиком
# в виджет не попадают: "Cannot switch to "+"a vessel"/"an object"+" we do not own.",
# длинные тела диалога установки (собираются с динамическим списком путей),
# суффиксы списка ". Currently active"/". Loaded" (склеиваются в "{0}. {1}{2}").
