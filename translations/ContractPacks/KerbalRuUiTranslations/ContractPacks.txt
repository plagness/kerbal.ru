# ContractPacks — UI-словарь СТАТИЧНЫХ отображаемых строк контрактов.
# Источник: .cfg-файлы паков CleverSat / Constellations / FieldResearch / KerbinSpaceStation
#   (GameData/ContractPacks/**), НЕ DLL. Извлечены значения полей title / description / notes /
#   synopsis / completedMessage / genericTitle / genericDescription у самих CONTRACT_TYPE и у
#   вложенных PARAMETER / REQUIREMENT (их title = цели в чек-листе контракта, что видит игрок),
#   плюс статичные строки DATA-пулов descs (descN, которые description = @/descs.Random()
#   показывает как есть).
#
# Движок (KerbalRuUiTranslator) матчит ТОЧНУЮ строку целиком (Ordinal), первый TAB — разделитель,
#   промах = строка идёт как есть (no-op). Поэтому ключи здесь — байт-в-байт значения из .cfg
#   (включая опечатки оригинала: varity, telescopee, planets's, procesing, goverment, THe, двойные
#   пробелы, «to to») — их править НЕЛЬЗЯ, иначе совпадения не будет.
#
# Взято ТОЛЬКО статичное. Намеренно НЕ включено (собирается в рантайме, точного совпадения не будет):
#   - Строки с токенами @ / <<..>> / {..} / % (ссылки, форматирование).
#   - 6 CC-выражений (кавычки + конкатенация/вызовы функций): title/description с "..."+HomeWorld()+"...",
#     'Kerbostationary orbit around HomeWorld() is possible', quoted "Target body must be a planet...".
#   - 5 многострочных значений (буквальный \n или TAB внутри) — не влезают в TAB-разделённый формат:
#     два больших description KSO-ретранслятора, notes про антенны, description KPS-прототипа, notes
#     станции с TAB ("...support 4 Kerbals.<TAB>This can be done...").
#   - Служебные заглушки: Dummy / Dummy part (Marker/dummy.cfg), cleanup (hidden=!DEBUG параметр).
#
# Термины: satellite=спутник, relay=ретранслятор (спутник-ретранслятор), orbit=орбита,
#   solar panel=солнечная панель, power generator=генератор энергии, antenna=антенна,
#   eccentricity=эксцентриситет, circularize=скруглить орбиту, optional=опционально, launch=запуск,
#   vessel=аппарат, body=небесное тело, kerbal=кербал, station-keeping=удержание позиции,
#   EVA=открытый космос (EVA). Латиницей: KSC, KSO/LKO/MKO, KPS, SENTINEL, RTG, TACLS, HiRes,
#   коды деталей (TH-NKR, D-ZZY, WT-SIT, F-RRY, MPL-LG-2, PPD-10) и названия модов/агентств-брендов
#   (CommTech, ComTech, Frontier Link, Duna One, Kerbstar, Station Parts Expansion, Tokamak
#   Industries, HabTech2, Tantares, Planetary Base Inc). «Ёлочки», ё.
#
# Формат: Английский<TAB>Русский. Несовпадение = строка остаётся как есть.
