# _common-ui -- общий словарь МЕТОК PAW (меню правого клика по деталям, KSP stock + моды).
# Игрок видит эти кнопки/подписи постоянно, в каждом полёте. Это НЕ один мод, а сквозной
# набор player-facing действий (Open/Close/Deploy/Retract/Toggle/Collect/Run/Review/Take/Log...),
# отрисовываемых через IMGUI/UGUI/TMPro -- перехватываются Harmony-патчами tools/ui-translator.
# folder="*" (не привязан к одному моду), деплоится всегда.
#
# Формат: Английский<TAB>Русский. Строки с # -- комментарии. Несовпадение = строка идёт как есть
# (no-op). Ключ матчится ПОБАЙТНО (Ordinal), включая кавычки: "Power On"/"Power Off" -- ключ и
# перевод намеренно сохраняют ASCII-кавычки (так метка отрисована в игре).
#
# ЕДИНАЯ ТЕРМИНОЛОГИЯ (канон для всего проекта, унифицирует разнобой per-mod файлов):
#   Deploy -> Развернуть, Retract -> Свернуть, Extend -> Выдвинуть, Toggle X -> Переключить X.
#   (Ранее bare Deploy/Retract в BahaSP/ROLib/DMagic/MechJeb2 были Выдвинуть/Втянуть --
#   приведены к этому канону, чтобы одна и та же кнопка звучала одинаково во всех модах.)
# АББРЕВИАТУРЫ: код прибора без общепринятого русского эквивалента остаётся латиницей
#   (GRS, RPWS, LUV, ELIX); имеющие устоявшуюся русскую форму -- переводятся (RTG->РИТЭГ, RW->маховик).
#   Gravioli -- игровой термин KSP, транслитерируется (гравиоли).
