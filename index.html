Что это

RO / RSS / RP-1 — флагманская реалистичная сборка KSP1. Она заменяет игрушечную кербальскую систему на настоящую Солнечную (Земля в реальном масштабе), делает физику, топливо и двигатели приближенными к жизни и добавляет историческую карьеру. Это признанный сообществом «тот самый» реалистичный режим.

kerbal.ru — русификатор поверх этой сборки. Не отдельный мод: он добавляет русский язык в 20 ключевых модов сборки и устанавливается копированием одной папки. Для англоязычных игроков ничего не меняется — перевод активируется только при выборе русского языка.

01 / СБОРКА

Шаг 1 · Ставим CKAN

Сборка ставится через CKAN — менеджер модов, который сам подтянет все зависимости. Сначала установи через Steam саму Kerbal Space Program (нужна версия 1.12.5), затем CKAN — по вкладке своей ОС:

🪟Windows
🍎macOS
🐧Linux
🎮Steam Deck
  1. Убедись, что установлен .NET Framework 4.8 — в Windows 10 и 11 он уже есть по умолчанию.
  2. Скачай ckan.exe со страницы релизов CKAN и положи в любую папку (не внутрь игры).
  3. Запусти ckan.exe двойным кликом — откроется графический интерфейс.
  4. При первом запуске укажи путь к установленной игре — CKAN обычно находит Steam-копию сам.
  1. Скачай CKAN.dmg со страницы релизов CKAN.
  2. Открой образ и перетащи CKAN.app в любую папку, затем запусти двойным кликом.
  3. На современных macOS (10.15+) CKAN открывается как текстовый интерфейс в Терминале — Apple убрала поддержку 32-битного GUI. Разреши доступ к Терминалу, если система спросит.
  4. Укажи путь к установленной игре и переходи к шагу 2.
Совет: навигация в консольном интерфейсе — стрелки, Enter, Tab. Либо собери сборку на ПК с Windows и просто скопируй папку игры на Mac.
  1. Установи Mono (движок для запуска CKAN): 
    # Debian / Ubuntu
sudo apt install mono-complete

# Fedora
sudo dnf install mono-complete

# Arch
sudo pacman -S mono
  2. Скачай ckan.exe со страницы релизов (либо ставь .deb/.rpm из тех же релизов).
  3. Запусти графический интерфейс: 
    mono ckan.exe
  4. Укажи путь к игре — обычно ~/.local/share/Steam/steamapps/common/Kerbal Space Program.

У SteamOS файловая система «только для чтения», поэтому чистый способ — контейнер Debian через distrobox (он уже есть на Deck). Всё делается в режиме рабочего стола в приложении Konsole.

  1. Создай контейнер и войди в него: 
    distrobox create -n ckanbox -i debian:12 -Y
distrobox enter ckanbox
  2. Внутри контейнера поставь Mono и CKAN: 
    sudo apt update && sudo apt install -y mono-complete
curl -L -o ckan.exe https://github.com/KSP-CKAN/CKAN/releases/latest/download/ckan.exe
  3. Подключи игру (домашняя папка видна из контейнера напрямую): 
    mono ckan.exe instance add --headless "KSP" \
"$HOME/.local/share/Steam/steamapps/common/Kerbal Space Program"
mono ckan.exe instance default KSP
mono ckan.exe update
  4. Дальше — команды установки из шага 2 (в том же контейнере).
Проще: если есть ПК — собери сборку там через CKAN и скопируй папку Kerbal Space Program на Deck целиком. Тогда контейнер не нужен.
02 / МОДЫ

Шаг 2 · Ставим RO / RSS / RP-1

В CKAN найди и отметь RP-1-ExpressInstall — это официальный мета-пак, который подтянет Realism Overhaul, Real Solar System, RP-1 и все зависимости. Через GUI: поставь галочки, нажми Apply. Через терминал (Linux / Steam Deck) — одной командой:

mono ckan.exe install --headless --no-recommends \
  RP-1-ExpressInstall RP-1-ExpressInstall-Graphics-Low RP-1
Graphics-Low выбран сознательно: низкое разрешение текстур экономит память и лучше идёт на слабом железе и Steam Deck. Первый запуск игры будет долгим — Module Manager собирает конфиги десятков модов.
03 / РУСИФИКАТОР

Шаг 3 · Ставим русский

  1. Скачай русификатор — ZIP-архив — и распакуй.
  2. Скопируй содержимое папки GameData из архива в GameData своей установки игры. Файлы лягут в нужные подпапки сами. Где находится игра:
    WindowsC:\Program Files (x86)\Steam\steamapps\common\Kerbal Space Program\GameData
    macOS~/Library/Application Support/Steam/steamapps/common/Kerbal Space Program/GameData
    Linux~/.local/share/Steam/steamapps/common/Kerbal Space Program/GameData
    Steam Deck~/.local/share/Steam/steamapps/common/Kerbal Space Program/GameData
  3. Включи русский язык. В игре: настройки → язык → Русский. Или вручную в файле settings.cfg (лежит рядом с папкой GameData): 
    LANGUAGE = ru
  4. Запусти игру. Готово — интерфейс, детали и карьера RP-1 на русском. 🚀

Что важно знать

  • Переведено 20 модов из полного набора сборки — остальные пока на английском. Список открыт для расширения.
  • Контракты, созданные до установки перевода в уже начатом сохранении, останутся на английском: их текст «запекается» в момент создания. Новые контракты будут на русском.
  • Несколько отдельных деталей (в ProceduralFairings и ModularLaunchPads) оставлены на английском — их исходные файлы устроены так, что нельзя было надёжно сверить перевод с оригиналом.
