Русская локализация для сборки Realism Overhaul · Real Solar System · RP-1
✧ ── 🚀 ── ✧
Это набор файлов локализации для реалистичной карьеры KSP1 на базе Realism Overhaul, Real Solar System и RP-1 (Realistic Progression One). Не отдельный мод — устанавливается поверх уже собранной сборки, копированием одной папки в GameData.
⬇ Скачать ZIP Репозиторий на GitHub

🚀 Установка

  1. Собери базовую сборку через CKAN: RP-1-ExpressInstall + RP-1-ExpressInstall-Graphics-Low + RP-1.
  2. Скачай этот репозиторий (кнопка выше) и распакуй архив.
  3. Скопируй содержимое папки GameData из архива в папку GameData своей установки KSP — файлы разложатся по нужным подпапкам автоматически.
  4. Выбери русский язык в настройках KSP (или напрямую в settings.cfg): 
    LANGUAGE = ru

⚠ Известные ограничения

  • Переведено 20 модов из полного набора большой сборки — остальные (~50) пока на английском.
  • Контракты, сгенерированные до установки перевода в уже существующем сохранении, останутся на английском — текст запекается в момент создания контракта. Новые контракты будут на русском.
  • Несколько отдельных деталей (в ProceduralFairings и ModularLaunchPads) не переведены — их исходные файлы содержат структурные особенности, из-за которых нельзя было надёжно сверить перевод с оригиналом.
