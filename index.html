00 / ОБЗОР

Что это

RO / RSS / RP-1 — флагманская реалистичная сборка KSP1. Она заменяет игрушечную кербальскую систему на настоящую Солнечную (Земля в реальном масштабе), приближает физику, топливо и двигатели к жизни и добавляет историческую карьеру. Признанный сообществом «тот самый» реалистичный режим.

kerbal.ru — русификатор поверх этой сборки. Не отдельный мод: он добавляет русский язык в 20 ключевых модов и ставится копированием одной папки. Для англоязычных игроков ничего не меняется — перевод включается только при выборе русского языка.

01 / СБОРКА

Шаг 1 · Ставим CKAN

Сборка ставится через CKAN — менеджер модов, который сам подтянет все зависимости. Сначала установи через Steam саму Kerbal Space Program (версия 1.12.5), затем CKAN — по вкладке своей ОС:

  1. Убедись, что установлен .NET Framework 4.8 — в Windows 10 и 11 он есть по умолчанию.
  2. Скачай ckan.exe со страницы релизов CKAN и положи в любую папку (не внутрь игры).
  3. Запусти ckan.exe двойным кликом — откроется графический интерфейс.
  4. При первом запуске укажи путь к игре — CKAN обычно находит Steam-копию сам.
  1. Скачай CKAN.dmg со страницы релизов CKAN.
  2. Открой образ, перетащи CKAN.app в любую папку и запусти двойным кликом.
  3. На современных macOS (10.15+) CKAN открывается как текстовый интерфейс в Терминале — Apple убрала поддержку 32-битного GUI. Разреши доступ к Терминалу, если система спросит.
  4. Укажи путь к игре и переходи к шагу 2.
Проще: собери сборку на ПК с Windows и просто скопируй папку игры на Mac — тогда возня с консолью не нужна.
  1. Установи Mono (движок для запуска CKAN): 
    # Debian / Ubuntu
sudo apt install mono-complete

# Fedora
sudo dnf install mono-complete

# Arch
sudo pacman -S mono
  2. Скачай ckan.exe со страницы релизов (либо ставь .deb/.rpm оттуда же).
  3. Запусти графический интерфейс: 
    mono ckan.exe
  4. Укажи путь к игре — обычно ~/.local/share/Steam/steamapps/common/Kerbal Space Program.

У SteamOS файловая система «только для чтения», поэтому чистый путь — контейнер Debian через distrobox (он уже есть на Deck). Всё делается в режиме рабочего стола, в приложении Konsole.

  1. Создай контейнер и войди в него: 
    distrobox create -n ckanbox -i debian:12 -Y
distrobox enter ckanbox
  2. Поставь Mono и CKAN внутри контейнера: 
    sudo apt update && sudo apt install -y mono-complete
curl -L -o ckan.exe https://github.com/KSP-CKAN/CKAN/releases/latest/download/ckan.exe
  3. Подключи игру (домашняя папка видна из контейнера напрямую): 
    mono ckan.exe instance add --headless "KSP" \
"$HOME/.local/share/Steam/steamapps/common/Kerbal Space Program"
mono ckan.exe instance default KSP
mono ckan.exe update
  4. Дальше — команда установки из шага 2 (в том же контейнере).
Проще: если есть ПК — собери сборку там и скопируй папку Kerbal Space Program на Deck целиком. Контейнер тогда не нужен.
02 / МОДЫ

Шаг 2 · Ставим RO / RSS / RP-1

В CKAN найди и отметь RP-1-ExpressInstall — официальный мета-пак, который подтянет Realism Overhaul, Real Solar System, RP-1 и все зависимости. Через GUI — галочки и Apply. Через терминал (Linux / Steam Deck) — одной командой:

mono ckan.exe install --headless --no-recommends \
  RP-1-ExpressInstall RP-1-ExpressInstall-Graphics-Low RP-1
Graphics-Low выбран сознательно: низкое разрешение текстур экономит память и лучше идёт на слабом железе и Steam Deck. Первый запуск игры будет долгим — Module Manager собирает конфиги десятков модов.
03 / РУСИФИКАТОР

Шаг 3 · Ставим русский

Автоматически — одной командой. Скрипт сам найдёт установку игры, скачает переводы и включит русский язык:

# Linux · macOS · Steam Deck
curl -fsSL https://kerbal.ru/install-ru.sh | bash
# Windows (PowerShell)
irm https://kerbal.ru/install-ru.ps1 | iex

Или вручную — в три шага:

  1. Скачай русификатор — ZIP-архив — и распакуй.
  2. Скопируй содержимое папки GameData из архива в GameData своей игры. Файлы лягут в нужные подпапки сами. Где находится игра:
    WindowsC:\Program Files (x86)\Steam\steamapps\common\Kerbal Space Program\GameData
    macOS~/Library/Application Support/Steam/steamapps/common/Kerbal Space Program/GameData
    Linux~/.local/share/Steam/steamapps/common/Kerbal Space Program/GameData
    Deck~/.local/share/Steam/steamapps/common/Kerbal Space Program/GameData
  3. Включи русский язык. В игре: настройки → язык → Русский. Или вручную в settings.cfg (рядом с папкой GameData): 
    LANGUAGE = ru
  4. Запусти игру. Перевод включится для интерфейса, карьеры и поддержанных деталей; фактическое покрытие каждого мода показано ниже.

Roadmap

  • Готово: переводы для 20 модов, установщики, сайт и проверяемый аудит через ModuleManager.ConfigCache.
  • Приоритет №1: поднять реальное покрытие названий выше текущих 36% и разобраться, почему часть корректных ключей не появляется в игре.
  • Дальше: 17 реальных кандидатов из полного состава, единый глоссарий терминов, установка через CKAN (netkan), поток контрибуций. Ещё 28 служебных компонентов перевода не требуют.

06 / НЮАНСЫ

Что важно знать

  • Русский блок есть у 38 из 83 компонентов сборки: 20 поддерживает kerbal.ru, ещё 17 модов и стоковая KSP поставляют его сами. Это охват состава, а не гарантия 100% строк.
  • Контракты, созданные до установки перевода в уже начатом сохранении, останутся на английском: их текст «запекается» в момент создания. Новые контракты будут на русском.
  • Проверка по совпадению ключей и исходных файлов недостаточна. Золотой тест — текст, который реально загрузился в ModuleManager.ConfigCache; именно его проценты показаны выше.
