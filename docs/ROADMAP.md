### 🚫 Сознательно НЕ переводим (не пробелы)

- Легитимные обозначения двигателей/ракет (RD-107, Merlin 1D, AJ-60A) и западные/вымышленные бренды
  (Aerojet, Rocketdyne, Alphadyne) — латиница по конвенции.
- Детали, физически удалённые из сборки (`HardRemoveNonRP0` и т.п.) — их «непереведённость» мнимая.
- Динамический UI-текст с числами/датами (склейка через `string.Format`/`Concat` — точное совпадение
  словаря невозможно по построению).
- Текст через `TMP_Text.SetText()` с многострочными `\n` — формат словаря построчный.
- `CommunityCategoryKit` вкладки категорий VAB — захардкожены без системы ключей, нужна правка мода.
