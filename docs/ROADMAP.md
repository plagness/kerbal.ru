Свежий `ModuleManager.ConfigCache` подтвердил: **56.8% title / 71.7% description / 13.9% manufacturer** по деталям (было 45.7%/63.4%/5.7% утром 22-го). Полная разбивка — [COVERAGE.md, раздел 0](COVERAGE.md#0-реальный-охват-по-деталям-партам--проверено-по-кэшу).

- [x] **SXT** — частично объяснено: 60 из 208 партов (наш bypass-патч) работают подтверждённо; для остальных 148 (родная локализация SXT, не сломана сама по себе) обнаружена системная причина низкого числа — RP-1 вырезает детали без `RP0conf=True` через собственный фильтр `HardRemoveNonRP0`, это не баг перевода. Подробности — [COVERAGE.md](COVERAGE.md#-2026-07-23-живой-запуск-подтвердил-фикс-final--sxtrealchute-загадка-решена-там-где-мы-её-патчили).

Оказалось: `RDNode` — не корневой тип узла, а вложенный в `@TechTree{}` — нужен вложенный патч `@TechTree:FINAL { @RDNode:HAS[#id[x]] {...} }`. Исправлено, ждёт следующей проверки.
- [x] **RealismOverhaul** — найдены ещё 2 собственных парта (`ROAdvCapsule`, `RO-OldFighterCockpit`), пропущенных вчерашней ручной сверкой; переведены, ждут подтверждения. Итого 8 подтверждённых собственных партов мода (был 1).
- [x] **Глоссарий производителей** — переведено ~262 ключа `#roMfr*` (RealismOverhaul/en-us-Mfr.cfg), использовался у сотен деталей по всей сборке без русской версии вообще.
- [x] **Большой проход, 12 параллельных агентов** — новые реальные пробелы закрыты: RealSolarSystem (358 ключей — флейвор-текст научных отчётов + описания тел), RP-1/Strategies (68 узлов, биографии исторических личностей с бонусами Admin-здания), B9_Aerospace_ProceduralWings (171 ключ), RealFuels (5 апгрейдов MLI). Подтверждены работающими в свежем кэше (Челомей, Программы и т.д.).

## 🟡 Обновление 2026-07-22 (вечер): исправление применено, ждёт подтверждения

Аудит того же дня утром показал, что «переведено 20 модов» означало «файлы написаны и структурно сверены», но **не** «реально показывается в игре» — фактический охват по деталям был **36% title, 54% description**. Вместо того чтобы искать причину (родная система локализации SXT/ROCapsules/VenStockRevamp/RealChute почему-то не резолвится, хотя всё определено верно), выбрали обходной путь: досрочно перевели недостающее и применили тем же надёжным доп-патчем (`%title = #LOC_kerbalru_...`), что уже даёт 98-100% у ModularLaunchPads/ROSolar. Причина исходной поломки так и не найдена — просто перестала быть нужна для прогресса.

- [x] Переведено и применено: SXT (+56 title/+53 desc), ROCapsules (+94/+92), VenStockRevamp (+25/+26), RealChute (+4/+4), ProceduralParts (+1), ProceduralFairings (+3/+2). Структурно проверено (длина слов, скобки, парность ключей) — итого **551/1023 title (53.9%), 726/1023 description (71.0%)**, если патч резолвится как ожидается.
- [ ] **Подтвердить живым запуском**: нужен один перезапуск KSP, чтобы ModuleManager пересобрал кэш, потом сверка по `ModuleManager.ConfigCache` (тот же тест, что поймал `Sputnik PS-1`). Пока цифры выше — проекция, не факт.
- [ ] Осталось (не входило в этот проход): ROEngines (~215 title, часть — легитимные designator-исключения), ROTanks (12, похоже на switcher-варианты, не отдельные парты), ROHeatshields (6, аналогично), RealismOverhaul (до 36, реальный масштаб не определён — нужна ручная сверка правок чужих партов через `@PART[stock]`).

Подробности и полная таблица — в [COVERAGE.md, раздел 0](COVERAGE.md#0-реальный-охват-по-деталям-партам--проверено-по-кэшу).

## 🟢 Обновление 2026-07-22 (ночь, после живого запуска): подтверждено кэшем

Пользователь перезапустил игру — сверка по свежему `ModuleManager.ConfigCache` дала реальные (не проекция) цифры: **45.7% title / 63.4% description** по деталям (было 36%/53.7% утром). Из работы по квестам/меню подтверждено 5 из 6 категорий; техдерево было полностью сломано из-за неправильного синтаксиса MM-селектора — найдено и исправлено, ждёт следующей проверки.

- [x] ROCapsules (95% title), ProceduralFairings (92%) — техника обходного патча подтверждена, почти полное покрытие.
- [x] Квесты (254 контракта), программы карьеры (35), Tracking Station (10), биомы (252), стартовые площадки (57) — **все подтверждены живым запуском**.
- [x] **Найдена и исправлена** причина полного провала техдерева (332 узла): синтаксис `@RDNode:HAS[#id[x]]` не матчил цель (MM распознавал патч, но не применял — 0 "Applying update" в логе). Исправлено на `@RDNode[*]:HAS[#id[x]]`, ждёт подтверждения.
- [ ] VenStockRevamp — частичный результат (26% вместо ожидаемых ~41%), причина не установлена.
- [x] **✅ Загадка SXT/RealChute решена.** RealismOverhaul сам патчит эти детали (и выборочные у ROTanks/ROSolar/ProceduralFairings/VenStockRevamp) в проходе `:FOR[RealismOverhaul]`, который выполняется позже нашего патча — RO затирает наш перевод обратно на английский. Фикс: `:FINAL` на все 718 заголовков `@PART[...]` во всех 12 файлах. Подробности — [COVERAGE.md](COVERAGE.md#-причина-исходной-поломки--решена-2026-07-22-ночь). Ждёт подтверждения следующим запуском.
- [ ] Осталось нетронутым: ROEngines (~215 title), ROTanks (12), ROHeatshields (6), RealismOverhaul (до 36).

Подробности — [COVERAGE.md, разделы 0 и 0b](COVERAGE.md#0-реальный-охват-по-деталям-партам--проверено-по-кэшу).

## 🎯 Следующие приоритеты (пересмотрено 2026-07-22, вечер — предыдущая версия этого раздела была ошибочной)

Проверка показала: из 17 модов раздела 3b **у 16 нет вообще никакого переводимого текста в `.cfg`** (весь UI зашит в `.dll`, наш метод не достаёт). Это касается и RealAntennas (заявленные «128 партов» были патчами к СТОКОВЫМ деталям, не своим — реальных строк оказалось всего 4, уже переведены) и Skopos (только технические радио-параметры, 0 игрового текста). Подробности и полный список проверенных модов — [COVERAGE.md, раздел 3b](COVERAGE.md#3b-️-пересмотрено-2026-07-22-у-большинства-из-этих-17-переводить-нечего).

**Вывод: структурно переводимый контент этой сборки почти исчерпан.** Реальный резерв — не новые моды, а:

1. **ROEngines — ~215 title.** Самый крупный настоящий остаток (часть — легитимные designator-исключения вроде `RD-0207`, реальный объём меньше, но всё равно самый большой из того, что действительно можно перевести).
2. **RealismOverhaul — до 36 партов неясного масштаба.** 30 из 36 не определяются в кэше нашим методом — нужна ручная сверка, возможно там реальных строк меньше, чем кажется.
3. ~~Загадка SXT/RealChute~~ — ✅ решена и исправлена (`:FINAL`), см. выше. Тот же фикс, вероятно, поднимет ROTanks/ROSolar/ProceduralFairings/VenStockRevamp ближе к 100% — RealismOverhaul патчит их выборочные детали тем же механизмом.
4. **ROTanks/ROHeatshields «missing from cache»** (12 и 6) — вероятно switcher/subtype-варианты, нужна ручная сверка, может оказаться что там нечего переводить (как с RealAntennas).
5. Долгосрочно, не для этого репозитория: PR/issue авторам plugin-модов (RealAntennas, TestFlight, CommunityCategoryKit и т.д.) с просьбой добавить поддержку `#LOC_`-ключей в их UI — реальный путь, но зависит не от нас.
