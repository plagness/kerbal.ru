# Быстрый старт — всё на лицензию

Для владельцев лицензионной Kerbal Space Program в Steam (версия 1.12.5).

---

## Способ А — один установщик на всё (проще всего)

Установи **Kerbal Space Program** через Steam (если ещё нет), затем одна команда:

- **Linux / macOS / Steam Deck** — в терминале:
  ```bash
  curl -fsSL https://kerbal.ru/install.sh | bash
  ```
- **Windows** — в PowerShell:
  ```powershell
  irm https://kerbal.ru/install.ps1 | iex
  ```

Если сборки RO/RSS/RP-1 ещё нет в `GameData`, установщик спросит, что делать (или сразу передай флаг):
`--full`/`-Full` — сам поставит сборку `RP-1-ExpressInstall` через официальный CKAN headless-режимом
(без диалогов и подтверждений) и сразу русификатор; `--ru-only`/`-RuOnly` — только русификатор, если
моды уже стоят или будут поставлены отдельно. Если сборка уже есть — выбирается только русификатор
автоматически, без вопросов. На Linux/macOS для шага CKAN нужен `mono` (на Steam Deck — внутри
`distrobox`, см. частые вопросы ниже); Windows запускает `ckan.exe` напрямую.

Готово — переходи сразу к концу этой страницы.

## Способ Б — по шагам вручную (если хочешь контролировать каждый этап)

### Действие 1 — поставить сборку

1. Установи **Kerbal Space Program** через Steam (если ещё нет).
2. Скачай **CKAN** — менеджер модов: <https://github.com/KSP-CKAN/CKAN/releases/latest>
   (Windows — `ckan.exe`; macOS — `CKAN.dmg`; Linux — `ckan.exe` через `mono`).
3. Открой CKAN, в поиске набери **`RP-1-ExpressInstall`**, поставь галочку и нажми **Apply**.
   CKAN сам скачает и разложит всю сборку: Realism Overhaul, Real Solar System, RP-1 и зависимости.

> Подробные шаги для каждой ОS — на сайте [kerbal.ru](https://kerbal.ru) → «Шаг 1».

### Действие 2 — поставить русский

**Автоматически (одна команда):**

- **Linux / macOS / Steam Deck** — в терминале:
  ```bash
  curl -fsSL https://kerbal.ru/install-ru.sh | bash
  ```
- **Windows** — в PowerShell:
  ```powershell
  irm https://kerbal.ru/install-ru.ps1 | iex
  ```
  Скрипт сам найдёт установку KSP, скачает переводы, скопирует их и включит русский язык.

Команда всегда выбирает последний стабильный GitHub Release. Её же нужно повторить после выхода новой версии: установщик сравнит `.kerbalru-version`, сделает backup заменяемых файлов и безопасно обновит только файлы kerbal.ru. Проверка без установки и откат описаны в **[UPDATING.md](UPDATING.md)**.

**Вручную (если не доверяешь скриптам):**

1. Скачай архив: <https://github.com/plagness/kerbal.ru/archive/refs/heads/main.zip>
2. Скопируй содержимое папки `GameData` из архива в `GameData` своей игры.
3. В настройках игры выбери язык **Русский** (или в `settings.cfg` строка `LANGUAGE = ru`).

При ручной установке версия и список управляемых файлов не записываются, поэтому автоматический откат недоступен. Для дальнейших обновлений удобнее хотя бы один раз запустить установщик.

Готово. Запускай игру — интерфейс, детали и карьера RP-1 на русском.

### Частые вопросы

- **Первый запуск очень долгий.** Это нормально: Module Manager собирает конфиги десятков модов. Дальше быстрее.
- **Контракты в старом сохранении на английском.** Их текст «запечён» в сейв при создании. Новые контракты будут на русском; на новой карьере всё сразу по-русски.
- **Часть деталей на английском.** Переведено 20 ключевых модов сборки, остальные — в планах (см. [ROADMAP](ROADMAP.md)).
- **Вышла новая версия перевода.** Закрой игру и снова выполни ту же команду установки. Подробности — в [UPDATING.md](UPDATING.md).
- **Steam Deck.** У SteamOS файловая система read-only — для CKAN нужен контейнер (`distrobox`) или сборка на ПК с копированием на Deck. См. сайт → вкладка «Steam Deck».
