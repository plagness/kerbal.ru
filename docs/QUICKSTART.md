# Быстрый старт — русская сборка KSP в пару команд

Для владельцев лицензионной Kerbal Space Program в Steam (версия **1.12.5**).

kerbal.ru — это **каталог готовых русских сборок + установщик** и **библиотека переводов модов**. Ниже — как поставить
сборку целиком. Хочешь только перевод одного мода — см. раздел в конце.

---

## Поставить сборку (менеджер сборок)

Установи **Kerbal Space Program** через Steam (если ещё нет), затем одна команда:

- **Linux / macOS / Steam Deck** — в терминале:
  ```bash
  curl -fsSL https://kerbal.ru/install.sh | bash            # покажет меню каталога
  curl -fsSL https://kerbal.ru/install.sh | bash -s -- --build operator   # сразу «Оператор»
  ```

Установщик — **менеджер сборок**: официальный CKAN скачает и разложит все моды сборки, поверх лягут наши переводы,
включится русский язык. При **повторном** запуске он находит уже стоящую сборку и предлагает **обновить / сменить /
только переводы**. Состояние хранится в `<KSP>/.kerbalru-build.json`.

Флаги: `--build <id>` · `--update` (обновить текущую) · `--ru-only` (только раскатать переводы) · `--list` (каталог) · `--yes`.

На Linux/macOS для шага CKAN нужен `mono` (на Steam Deck — внутри `distrobox`, см. вопросы ниже).

### Какие сборки есть

| id | Что это | Система | Сложность |
|----|---------|---------|-----------|
| **`operator`** | «Ванила+»: спутники → сеть покрытия → станции → автоматика (kOS). Долгая игра на стоке. | сток | 4/5 |
| **`rp1`** | RO/RSS/RP-1 — реальная Солнечная система, реальная физика, историческая карьера. Хардкор-реализм. | RSS | 5/5 |

Полный состав каждой — в `builds/<id>/build.json`; список — `--list` или на сайте [kerbal.ru](https://kerbal.ru).

## Только перевод одного мода (продвинутым)

Все переводы лежат по папке на мод в `translations/<mod>/`. Скопируй нужную папку к себе в `GameData/`
(Localization-файлы работают из любого места; для модов с UI-словарём нужен ещё `engine/Plugins/KerbalRuUiTranslator.dll`)
и выставь `LANGUAGE = ru` в `settings.cfg`.

## Частые вопросы

- **Первый запуск очень долгий.** Это нормально: Module Manager собирает конфиги десятков модов и дерево техов. Дальше быстрее.
- **Контракты в старом сохранении на английском.** Их текст «запечён» в сейв при создании. Новые контракты будут на русском;
  на новой карьере всё сразу по-русски.
- **Часть деталей на английском.** Переводим по модам постепенно; актуальный охват — `translations/_index.json` (или см. [ROADMAP](ROADMAP.md)).
- **Вышла новая версия.** Закрой игру и выполни `install.sh --update`. Проверка и откат — в [UPDATING.md](UPDATING.md).
- **Steam Deck.** У SteamOS файловая система read-only — для CKAN нужен контейнер `distrobox` (рецепт в [FOR-AGENTS.md](FOR-AGENTS.md))
  или собери сборку на ПК и скопируй папку игры на Deck целиком.
